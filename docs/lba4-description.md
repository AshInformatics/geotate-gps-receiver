# LBA 4

This is used to query or set various capture settings

Configurability of certain settings depend on the corresponding capture capabilty flag see (lba1-description.md)

## Important bytes

|Bytes | Description|
|--- | --- |
| 0-3  | Interval
| 4 | Quality
| 5 | Mode
| 6-9 | Continous mode
| 10-11 | Delay
| 12-13 | No-motion interval
