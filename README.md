# Geotate documentation and code

This is a collection of information and code surrounding the Geotate-based devices
Bilora GT-01 and JOBO photoGPS.

Documentation of the data and on-device data is available in [docs](docs). Code to
download the captured data from the device is in [geotate-downloader](geotate-downloader).
To run the JAVA application on Linux or without server connectivity, there are stubs for the JNI
code in [jni-stubs](jni-stubs).
